Human-Machine Collaborate Search Model

This is a joint project with designated responsible institutions of Shanghai International Studies University and Tsinghua University.
Project aimed to develop a search model for text data generated on media, in a human-machine interaction scenario, where the search results will be fine tuned after each human's collaboration.
I worked as the algorithm leader, Designed and implemented algorithm for the model. The algorithm mainly contains TF-IDF, Word2Vec for text vectorization, a LDA method for text clustering, and cosine similarity for fine tuning after human interaction. The test for the algorithm is designed with perplexity and cross validation.
I also implemented a GUI for human-computer interaction.
